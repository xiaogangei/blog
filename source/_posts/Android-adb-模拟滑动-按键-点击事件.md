我们有时候在手机上会有大量重复的操作，我们可以通过ADB的指令来模拟我们手指的点击与滑动，可以大大的提升我们的效率
<!-- more -->
### input命令的使用格式
Android的键盘和点击模拟时间全部靠input命令来完成，看一下input命令的格式：
``` BASH
usage: input ...

       input text <string> 向输入框输入文字
       input keyevent <key code number or name> 模拟键盘事件
       input tap <x> <y> 模拟触摸的位置
       input swipe <x1> <y1> <x2> <y2> 模拟滑动的位置从xy1到xyz2
```
### 示例
``` BASH
   input text "hello"
   input keyevent 3
   input tap 30 50
   input swipe 30 50 30 800
```
